# skills

A collection of open agent skills. Each subdirectory contains a `SKILL.md` with instructions for a specific capability.

## Available skills

### opensrc

Fetch npm package or GitHub repository source code to gain implementation-level context about how libraries work internally.

```bash
opensrc <package-name>
opensrc owner/repo
```

See [opensrc/SKILL.md](./opensrc/SKILL.md) for full usage instructions.
